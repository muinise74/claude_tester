필수 요구사항

🟢
Node.js 필수
v18 이상 설치 필요. Puppeteer 21.x 및 Electron 실행에 사용됩니다.
설치 확인: node -v
다운로드: https://nodejs.org
🤖
Claude CLI 필수
Anthropic에서 제공하는 CLI 도구. 테스트 코드 생성에 사용됩니다.
설치 확인: claude --version
설치 방법: npm install -g @anthropic-ai/claude-code
🌐
Claude for Chrome 활성화 필수
Claude CLI의 --chrome 플래그 기능. 활성화되어 있지 않으면 코드 생성 시 오류가 발생합니다.
활성화 방법: Claude CLI 실행 후 /chrome 명령어 입력하거나 claude --chrome 으로 실행
📦
Puppeteer 21.x 필수
앱 설치 시 npm install로 자동 설치됩니다. 생성된 테스트 코드 실행에 사용됩니다.
설치 확인: node -e "require('puppeteer')"

스킬 설정

🗂️
test-generator 스킬 등록 필수
Claude CLI가 테스트 코드를 생성할 때 사용하는 스킬입니다.
스킬 탭에서 저장 버튼을 눌러 Claude CLI 스킬 경로에 저장하세요.
저장 경로: ~/.claude/skills/test-generator/SKILL.md

권장 환경

💻
운영체제 권장
Windows 10/11, macOS 12 이상 권장.
Claude for Chrome 기능은 Chrome 브라우저가 설치되어 있어야 합니다.
💳
Anthropic Pro 구독 권장
Claude CLI 사용량이 많을 경우 Pro 플랜을 권장합니다.
무료 플랜은 사용량 제한으로 인해 코드 생성 도중 중단될 수 있습니다.
